## About

Github Profiles is an application for show dev profiles on Github.

This Project was developed for Bootcamp in Digital Innovation One.

## Technologies

- ReactJS
- Typescipt
- Normalize
- Styled Components

## Run Project

Clone Project

```git
git clone https://github.com/thiagotrs/github-profiles.git
```

RUN

```ssh
cd github-profiles
npm install
npm start
```

URL
```
http://localhost:3000/
```
